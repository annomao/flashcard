Javascript (JS)

is a scripting language that enables you to create dynamically updating content, control multimedia, animate images, and pretty much everything else.

Operators

there are various kinds:
assignment operators : = , +=, -=, *=, /=, %=, **=
Comparison operators: ==, ===, !=, !==, ?
Logical operators: &&, ||, !

Variables

Variables are containers for storing data (storing data values).
are declared using the let, var and const keywords

Variable naming conventions

JavaScript programmers tend to use camel case that starts with a lowercase letter e.g firstName, lastName A JavaScript name must begin with:
A letter (A-Z or a-z)
Or an underscore (_)
Subsequent characters may be letters, digits, underscores, or dollar signs. Numbers are not allowed as the first character in names.
hyphens (-) are reserved for substraction.

Functions

A JavaScript function is a block of code designed to perform a particular task. A JavaScript function is executed when "something" invokes it (calls it).
A JavaScript function is defined with the function keyword, followed by a name, followed by parentheses ().

Strings

A JavaScript string is zero or more characters written inside quotes.
You can use single or double quotes
let name = "Jane Doe"
let name2 = 'Jane Doe'

Methods

JavaScript methods are actions that can be performed on objects.
Methods are functions stored as object properties.
You access an object method with the following syntax:
objectName.methodName()

Arguments

Function arguments are the values received by the function when it is invoked

Parameters

Function parameters are listed inside the parentheses () in the function definition
function name(parameter1,parameter2,parameter3){
}

Return

The return statement ends function execution and specifies a value to be returned to the function caller. When JavaScript reaches a return statement, the function will stop executing.

Chaining Methods

Method chaining is the mechanism of calling a method on another method of the same object. This ensures a cleaner and readable code. Method chaining uses this keyword in the object's class to access its methods. In javascript, the this keyword refers to the current object in which it is called.

Booleans

A JavaScript Boolean represents one of two values: true or false.

Undefined

In JavaScript, a variable without a value, has the value undefined
let box ;
Any variable can be emptied, by setting the value to undefined
car = undefined

NaN

n JavaScript, NaN is short for "Not-a-Number".
In JavaScript, NaN is a number that is not a legal number.
The Global NaN property is the same as the Number.

Alert

An alert box is often used if you want to make sure information comes through to the user.
When an alert box pops up, the user will have to click "OK" to proceed.
alert("type something")

Comments

comments are meaningful way to deliver message. It is used to add information about the code, warnings or suggestions so that end user can easily interpret the code.
Single line comments start with //.
Multi-line comments start with /* and end with */.

JQuery

jQuery is a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers

Attributes

Attributes define additional characteristics or properties of the element such as width and height of an image. Attributes are always specified in the start tag (or opening tag) and usually consists of name/value pairs like name="value" . Attribute values should always be enclosed in quotation marks.

Prompt

A prompt box is often used if you want the user to input a value before entering a page.
When a prompt box pops up, the user will have to click either "OK" or "Cancel" to proceed after entering an input value.
If the user clicks "OK" the box returns the input value. If the user clicks "Cancel" the box returns null.
prompt("sometext","defaultText");
